# Machine Learning & Data Science Portfolio

This repository contains machine learning, computer vision, and data 
analysis projects built using Python. Each project includes a complete 
pipeline: data preprocessing, EDA, model training, and performance evaluation.

### Projects Included:
- Facial Recognition System (OpenCV, ML)
- Rainfall Prediction Model (Regression)
- Community powered music discovery  engine 
- Astronomical Data EDA

### Tools & Technologies:
Python, Scikit-Learn, OpenCV, Pandas, NumPy, Matplotlib, Plotly, Google Colab

# Facial Recognition System
This project implements a facial recognition system using Python and OpenCV.  
It detects and identifies faces in real-time using image preprocessing,  
feature extraction, and machine learning-based classification.
### Skills Demonstrated:
OpenCV, Python, Image Processing, ML Classifiers

# Rainfall Prediction Model
A machine learning model designed to predict rainfall using historical  
weather data. The project includes EDA, feature engineering, regression  
modeling, and performance evaluation.
### Skills Demonstrated:
Python, Pandas, Scikit-Learn, Matplotlib, Regression Modeling, EDA

# Astronomical Data Exploration (EDA)
A data analysis project focused on exploring astronomical observations  
using Python. Highlights include data cleaning, pattern detection,  
visualizations, and insights on celestial object distributions.
### Skills Demonstrated:
Python, Pandas, Visualization, EDA, Scientific Data Analysis
