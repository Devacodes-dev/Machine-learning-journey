# Machine Learning & Data Science Portfolio

This repository contains machine learning, computer vision, and data 
analysis projects built using Python. Each project includes a complete 
pipeline: data preprocessing, EDA, model training, and performance evaluation.

### Projects Included:
- Facial Recognition System (OpenCV, ML)
- Rainfall Prediction Model (Regression)
- Retail Sales Trend Analysis (EDA)
- Astronomical Data EDA

### Tools & Technologies:
Python, Scikit-Learn, OpenCV, Pandas, NumPy, Matplotlib, Plotly, Google Colab
